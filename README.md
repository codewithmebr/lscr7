# lscr7
lscr7 is a program that replaces the ls command with lscr7, displaying a customized file listing in the terminal.

## Requirements

- Linux
- Shell environment (bash, zsh, etc.)

## Installation

Clone the repository:

```bash
git clone https://github.com/seu-usuario/lscr7.git
cd lscr7
```

### Install chafa

Link: https://hpjansson.org/chafa/download/

## Run 
chmod +x script.sh
./lscr7.sh 'ls -l'
